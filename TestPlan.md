# Test Plan – Sauce Demo E-Commerce Website

## 1. Objective
Verify that core user flows including login, inventory browsing, and checkout function correctly and meet expected behaviour.

## 2. Scope

**Features tested:**
- Login (valid and invalid credentials)
- Inventory (product browsing, sorting, adding to cart)
- Checkout (form completion, order confirmation)

**Out of scope:**
- Payment processing
- Account registration
- Mobile responsiveness

## 3. Test Types
- Functional testing
- UI / usability testing

## 4. Test Environment
- Browser: Google Chrome (latest)
- OS: Windows 11
- URL: https://www.saucedemo.com
- Test account: standard_user

## 5. Entry Criteria
- Site is accessible and loads correctly
- Test credentials are valid

## 6. Exit Criteria
- All test cases executed
- All bugs documented in BugReports.md
